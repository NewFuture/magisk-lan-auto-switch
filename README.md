# Magisk Modules

This repository contains Magisk modules for various purposes. Each module is located in its own subdirectory.

## Modules

### [Magisk LAN Auto Switch](./magisk-lan-auto-switch/README.md)
Automatically switch LAN and WiFi based on `eth0` connection status.

## Installation
1. Navigate to the specific module's directory (e.g., `magisk-lan-auto-switch`).
2. Follow the installation instructions provided in the module's `README.md`.
